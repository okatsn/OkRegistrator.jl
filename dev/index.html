OkRegistrator.baseexactexpr — Method

baseexactexpr(regrepo) returns expression matching exactly the basename of regrepo.

Example

expr = OkRegistrator.baseexactexpr(".julia/dev/OkRegistry")
occursin(expr, "OkRegistry")

# output

true
occursin(expr, "NotOkRegistry")

# output

false
occursin(expr, "NotOkAtRegistryAll")

# output

false

OkRegistrator.isprerelease — Method

Given the package directory pkgpath, isprerelease(pkgpath) returns true if there is nothing left after matching r"(\d+.?)+". Use this to check whether to registor a version or not.

source
OkRegistrator.isprerelease — Method

isprerelease(ver::VersionNumber)

Example

isprerelease(v"0.99.5")

# output

false
isprerelease(v"0.99.5-DEV")

# output

true

OkRegistrator.okciregister — Method

Given the directory of your local registry okregpath, okciregister(okregpath) attempts to register all pacakges (folders) under dirname(okregpath).
